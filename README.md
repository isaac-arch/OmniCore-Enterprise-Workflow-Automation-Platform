1. REQUIREMENTS
- XAMPP (PHP 8+)
- Web browser
- Basic PHP/MySQL knowledge

2. INSTALLATION
- Open XAMPP Control Panel
- Start Apache and MySQL
- Go to htdocs folder
- Create folder named omnicore
- Paste all project files inside

3. DATABASE SETUP
- Open http://localhost/phpmyadmin
- Create database named omnicore
- Import database.sql file

4. RUNNING THE PROJECT
- Open browser
- Visit http://localhost/omnicore
- Register a new user
- Login to access dashboard

5. BACKGROUND JOBS
- Visit http://localhost/omnicore/worker/worker.php
- Refresh to process queued jobs
- Can be automated using cron

6. PROJECT FEATURES
- User authentication
- Project & task management
- Background job queue
- Activity logging
- Admin dashboard foundation

7. EXTENSIONS
- Role permissions
- Analytics charts
- API endpoints
- Automation rules
- SaaS billing

8. DEPLOYMENT
- Can be moved to live server
- Update db credentials
- Secure authentication
